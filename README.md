## 🇬🇧 English Documentation

**VOID.STUDIO** is a modern, high-performance web agency landing page built with a focus on minimalism and brutalism aesthetics. The project showcases services, pricing, and project portfolios with smooth animations and a bold design system.

### 🚀 Tech Stack

- **Framework:** React 19 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (v4)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **State Management:** Zustand
- **Deployment:** GitHub Pages (`gh-pages`)

### 📂 Project Structure

- `src/components/Hero.tsx` - Main introductory section with bold typography and animations.
- `src/components/Pricing.tsx` - Pricing plans (Landing & Corporate packages) and work process steps.
- `src/components/Upsells.tsx` - Additional services and features available to clients.
- `src/components/ContactForm.tsx` - Form for lead capture and project inquiries.
- `src/components/Projects.tsx` - Portfolio section showcasing past work.
- `src/components/Header.tsx` & `src/components/Footer.tsx` - Navigation and footer elements.

### 🛠️ Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```

4. **Deploy:**
   ```bash
   npm run deploy
   ```

### 💅 Design Philosophy
The design relies on high contrast (black and white), large typography, visible borders (brutalism), and micro-animations for an engaging user experience.

## 🇷🇺 Документация на русском

**VOID.STUDIO** — это современный, высокопроизводительный лендинг веб-студии, созданный с акцентом на эстетику минимализма и брутализма. Проект представляет услуги, цены и портфолио с плавными анимациями и смелой дизайн-системой.

### 🚀 Технологический стек

- **Фреймворк:** React 19 + TypeScript
- **Сборщик:** Vite
- **Стилизация:** Tailwind CSS (v4)
- **Анимации:** Framer Motion
- **Иконки:** Lucide React
- **Управление состоянием:** Zustand
- **Деплой:** GitHub Pages (`gh-pages`)

### 📂 Структура проекта

- `src/components/Hero.tsx` — Главный экран с крупной типографикой и анимациями.
- `src/components/Pricing.tsx` — Тарифные планы (Лендинг и Корпоративный) и этапы работы.
- `src/components/Upsells.tsx` — Дополнительные услуги для клиентов.
- `src/components/ContactForm.tsx` — Форма для сбора заявок и связи.
- `src/components/Projects.tsx` — Блок с примерами работ (портфолио).
- `src/components/Header.tsx` & `src/components/Footer.tsx` — Навигация и подвал сайта.

### 💅 Философия дизайна
Дизайн опирается на высокий контраст (черное и белое), крупную типографику, видимые границы элементов (брутализм) и микроанимации для вовлечения пользователя.
